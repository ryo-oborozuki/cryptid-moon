【画像ファイルの置き方】

このフォルダに以下のファイル名で画像を置くと、自動的にサイトに反映されます。
PNG / JPG / WEBP など、お好みの形式でOKです。
ファイルを置いたらブラウザを F5 でリロードするだけで切り替わります。

■ トップページ（Hero）
  title-logo.png
    → サイトタイトル「Cryptid Moon」のロゴ画像
    → 推奨: 横長・背景透過PNG（例: 800×200px）
    → ※ 置かない場合はテキスト「Cryptid Moon」が表示されます

  hero-concept.png
    → トップのイメージイラスト・コンセプトアート
    → 推奨サイズ: 横長（16:6 比率、例: 1600×600px）
